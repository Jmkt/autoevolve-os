# 🤝 Guía de Contribución

> Gracias por tu interés en contribuir al proyecto!

---

## 🎯 Cómo Contribuir

### Pruebas y Feedback
- Usa el sistema
- Reporta problemas
- Sugiere mejoras

### Documentación
- Mejora los prompts
- Añade ejemplos
- Traduce a otros idiomas

### Código
- Implementa nuevas características
- Corrige bugs
- Optimiza rendimiento

## 🚀 Primeros Pasos

```bash
# Clonar
git clone https://github.com/Jmkt/autoevolve-os.git
cd autoevolve-os

# Instalar
uv sync
```
