# 🧬 AutoEvolve OS - Infrastructure

> **El sistema operativo de IA que evoluciona automáticamente**
> **Versión:** 1.0 (Open Beta)
> **Fecha:** 2026-04-06

---

## 🎯 La Visión

Este proyecto explora un concepto nuevo en IA: **un sistema operativo donde la inteligencia artificial se mejora a sí misma continuamente**, sin intervención humana constante.

## 🔑 Características Principales

### 1. Investigación Autónoma
- Ciclos de 5 minutos
- Métricas claras de evaluación
- Optimización continua

### 2. Filtro Multi-Modelo
- Selección automática del mejor modelo
- Optimización de parámetros
- Balance costo/rendimiento

### 3. Sistema de Membrana
- 10 prompts interconectados
- eacha retroalimentación positiva
- Auto-mejora continua

### 4. Knowledge Graph Persistente
- Wiki que crece automáticamente
- Conexiones inteligentes
- Documentación autónoma

## 🚀 Quick Start

```bash
# Clonar
git clone https://github.com/Jmkt/autoevolve-os.git

# Instalar dependencias
uv sync

# Ejecutar
bash scripts/evolve.sh status
```

## 🎮 Comandos

| Comando | Función |
|---------|---------|
| `evolve status` | Ver estado del sistema |
| `evolve research` | Iniciar ciclo de investigación |
| `evolve optimize` | Optimizar componentes |
